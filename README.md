# vCard TUI

Terminal UI application for managing vCard contacts, written in Rust. It uses the vCard format for importing and exporting contact data.

## Install

Install via cargo.
```shell
cargo install vcard_tui
```

Install via homebrew.
```shell
brew tap kenianbei/vct
brew install vct
```

## Usage

```shell
vct
```
